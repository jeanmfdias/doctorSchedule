# DoctorSchedule

## Requirements

- PHP >= 7.1
- MySQL >= 5.6
- Node.js >= 8

## Steps to execute

1. `composer install`
2. `cp .env.example .env`
3. `php artisan key:generate`
4. Create database doctor_schedule_db
5. `php artisan migrate`
6. `npm install`
7. `npm run dev`
8. `php artisan serve`
10. Project running at http://127.0.0.1:8000
